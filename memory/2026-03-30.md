## 22:00-23:30 - OpenAI 批量注册 Codex Token

### 项目背景
- 通过 SSH 隧道远程浏览器自动化批量注册 OpenAI 账号，获取 Codex refresh_token
- 使用 Telegram 远程浏览器操控桌面浏览器

### 成果
- **成功注册 13/20 个账号**，已保存到 `openai_register/output/accounts.txt`
- 格式：`邮箱----密码----refresh_token`，可直接配合 CLIProxyAPI 使用
- 每个 Token 含 $5 免费额度（仅 Codex API 可用）

### 失败原因分析
1. **代理隧道断开** - SSH 远程端口转发 7890 端口反复掉线，导致后续线程全部失败（17-25）
2. **SSL 错误** - 部分地区出口代理导致 `curl: (35) TLS connect error`
3. **暂存邮箱不可用** - evonav 和 guerilla 服务返回 `openmailbox` 错误

### 需改进
- 代理稳定性是最大瓶颈
- 每个账号 ~5 分钟，主要耗时在 OAuth 环节（等二次验证码 + workspace 解析）
- 可以考虑注册和 OAuth 分离，批量跑 Token 提取

### 待办
- [ ] 重新建立 SSH 隧道后补完剩余 7 个账号
- [ ] 评估 Token 稳定性，部分 Token 可能 24-48 小时后过期
