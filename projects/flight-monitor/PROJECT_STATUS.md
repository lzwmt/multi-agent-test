## ✅ 已完成

### 1. 携程 (Ctrip) - 主数据源
**状态：✅ 完全可用**

| 功能 | 状态 | 说明 |
|------|------|------|
| 动态价格 | ✅ | 不同日期价格不同 |
| 航班号 | ✅ | 如 9C7515, CZ6180 |
| 起飞时间 | ✅ | 如 20:10 |
| 降落时间 | ✅ | 如 22:40 |
| 飞行时长 | ✅ | 如 2h30m |

**示例数据：**
```json
{
  "flightNo": "CZ3251",
  "depTime": "07:35",
  "arrTime": "10:40",
  "price": 400,
  "depCity": "广州",
  "arrCity": "青岛"
}
```

**调用方式：**
- Cookie 登录
- Playwright 抓取
- 稳定可靠

### 2. 去哪儿 (Qunar) - 辅助数据源
**状态：⚠️ 部分可用**

#### ✅ getAsyncPrice - 价格接口
| 功能 | 状态 | 说明 |
|------|------|------|
| 动态价格 | ✅ | 不同日期价格不同 |
| 指定航线 | ✅ | CAN_TAO |
| 航班号 | ❌ | 无 |
| 时间 | ❌ | 无 |

**示例数据：**
```json
{
  "flightCode": "CAN_TAO_2026-03-14",
  "price": "869"
}
```

**价格对比：**
| 日期 | 携程 | 去哪儿 |
|------|------|--------|
| 03-14 | ¥400 | ¥869 |
| 03-18 | ¥400 | ¥319 |

#### ❌ touchInnerList - 完整数据接口
| 功能 | 状态 | 说明 |

| 动态价格 | ✅ | 有 |
| 航班号 | ✅ | 有 |
| 时间 | ✅ | 有 |
| 指定航线 | ❌ | 需要 Bella 签名 |

## 🎯 推荐方案

### 方案A：携程为主（推荐）
```
主数据：携程
  - 价格
  - 航班号
  - 时间

辅助：AirLabs
  - 时间验证
```

**优点：**
- ✅ 数据完整
- ✅ 稳定可靠
- ✅ 开发成本低

### 方案B：携程 + 去哪儿价格
```
主数据：携程
  - 价格
  - 航班号
  - 时间

辅助：去哪儿(getAsyncPrice)
  - 额外价格参考
```

**优点：**
- ✅ 价格对比
- ✅ 发现更低价

### 方案C：多源整合
```
携程：主数据
去哪儿：价格参考
AirLabs：时间验证
```

**优点：**
- ✅ 数据最全面
- ⚠️ 复杂度高
