# GUI 开发计划 - Textual 终端界面

## 目标
为个人效率工具集添加美观现代的 Textual TUI 界面

## 功能模块

### 1. 主界面
- 侧边栏导航（任务 / 番茄钟 / 笔记）
- 主内容区动态切换
- 状态栏显示当前时间/统计

### 2. 任务管理界面
- 任务列表（带优先级颜色标识）
- 添加任务弹窗
- 完成任务勾选
- 删除任务确认
- 筛选：全部 / 未完成 / 已完成

### 3. 番茄钟界面
- 大字体倒计时显示
- 开始/暂停/重置按钮
- 进度环动画
- 今日统计
- 历史记录列表

### 4. 笔记界面
- 笔记卡片列表
- 标签云筛选
- 搜索框
- 添加/编辑笔记弹窗
- 删除确认

## 技术方案
- textual >= 0.40.0
- 复用现有 storage.py 数据层
- 保持 CLI 功能兼容

## 文件结构
```
src/
  main.py          # 原 CLI 入口
  main_tui.py      # TUI 入口
  tui/
    __init__.py
    app.py         # 主应用
    screens/       # 各功能屏幕
    widgets/       # 自定义组件
```

## 步骤
1. 安装 textual
2. 创建 TUI 应用框架
3. 实现任务管理界面
4. 实现番茄钟界面
5. 实现笔记界面
6. 测试 & 优化
